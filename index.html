HTML

  • The head element contains information about the webpage.
  • The body element represents the visible contest shown to the user.
  • The img element contains information about images that are displayed on the webpage.
  • The src attribute defines the location of the image.
  • The alt attribute contains a text string that describes the image and its functionality
